# QRes OS 4 - Панель управления

Панель управления для системы QRes OS 4 - инструмент для администрирования ресторана.

## Функции
- Управление персоналом
- Настройка меню
- Аналитика и отчеты
- Управление столиками и зонами
- Настройки системы

## Запуск
```bash
npm install
npm run dev
```

## Сборка
```bash
npm run build
```
